## 🎯 Overview

This repository contains the source code for my personal portfolio website. Designed with a developer-centric approach, it features a clean terminal-inspired aesthetic to highlight my expertise in backend technologies, data processing, and system architecture.

## 🚀 Tech Stack

- **Framework:** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** CSS / Tailwind CSS
- **Deployment:** [Vercel](https://vercel.com/)
- **Icons:** [Lucide React](https://lucide.dev/)

## 💻 Running the Project Locally

If you want to run this project on your local machine, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone [https://github.com/pitersoniko/portfolio-web.git](https://github.com/pitersoniko/portfolio-web.git)
   ```

2. Navigate to the project directory:
   ```bash
   cd portfolio-web
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and visit `http://localhost:5173` to view the project.
